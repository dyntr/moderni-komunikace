# Moderní komunikace — Webdesign studio

Webová prezentace studia **Moderní komunikace** — tvoříme moderní weby na míru, rychlé, responzivní a optimalizované pro růst vašeho podnikání.

## Technologie

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Instalace a spuštění

```bash
npm install
npm run dev
```

## Build pro produkci

```bash
npm run build
```

Výstup se vygeneruje do složky `dist/`.

## Deploy

Projekt je optimalizovaný pro nasazení na Cloudflare Pages.
