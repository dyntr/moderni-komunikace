## Overview

Marketing site for **Moderní komunikace**, a webdesign studio building custom websites for small and mid-size businesses. The site is the studio's own showcase: real client work, service breakdown, pricing, and a clear process — all built with the same React/Tailwind stack the studio sells to clients.

## ✦ Features

- **Filterable portfolio grid** — 7 real client projects (corporate sites, an e-shop, landing pages) tagged by type, with a live-preview modal that loads the actual site in an iframe
- **Services grid** — web builds, redesigns, landing pages, e-shop solutions, SEO & performance, ongoing maintenance
- **5-step process timeline** — consultation → design → development → launch → ongoing care
- **3-tier pricing** — Start / Business / custom Premium
- FAQ accordion, animated background blobs, floating elements, animated stat counters
- Contact form via Web3Forms

## 🛠 Built with

React 18 + TypeScript · Vite · Tailwind CSS · shadcn/ui (Radix primitives) · Framer Motion · React Hook Form + Zod · Vitest

## 📦 Getting started

```bash
npm install
npm run dev      # local dev server
npm run build    # production build → dist/
```

Optimized for deployment on Cloudflare Pages.
